Equation spotlight!

s = ut + 1/2 at²

What does each variable mean?!

s = Displacement (m).

u = Initial velocity (m/s).

t = Time elapsed (initial-scale).

a = Constant acceleration (m/s²).

Fun Fact About the Equation!

Did you know that the famous physics equation s = ut + ½at² is actually like a time machine in disguise? Mathematically, it works perfectly backward. If you plug in a negative time (-t), it reveals where an object would have been before the chosen reference time, assuming constant acceleration. While Galileo laid the groundwork for this formula by rolling bronze balls down wooden ramps, Sir Isaac Newton later connected it to calculus, proving that the "½" comes naturally from integration. Today, this equation is used in physics engines for games like Angry Birds and Rocket League to calculate realistic projectile motion.
